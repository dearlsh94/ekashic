자주 사용하는 VSCode IDE 단축키 24개

매체 속 컴퓨터하는 모습을 보면, 초보자는 마우스를 천천히 움직이고 전문가는 키보드를 빠르게 타닥거리는 모습으로 자주 묘사되는 것 같습니다. 역시 프로그래머의 꽃은 빠르게 키보드를 두드리는 것만으로 무언가를 만드는 모습이죠. (아님)

단축키가 익숙하지 않아 다소 불편하더라도 하던대로 하는 것보다, 익숙하지 않은 단축키라도 처음 몇 번 의식적으로 사용해보면서 천천히 손으로 기억해나간다면, 두고두고 유용하게 사용하며 개발 시간을 많이 줄여줄 수 있습니다. 단축키로 개발하는 멋진 내 모습을 느낄 수 있는 것은 덤이지요.

저는 vim과 터미널에 익숙하지는 않지만, 그래도 VSCode로 작업하면서는 단축키를 자주 사용합니다. 대부분 무의식으로 사용해서, 어떤 기능을 위한 손가락 모습은 취할 수 있지만 무슨 키를 눌러야되냐고하면 잘 떠오르지 않고는 해요.

그래서 이번에 제가 개발하면서 어떤 단축키들을 많이 사용하는 지 정리해보았습니다. 일부 생각이 나지 않아 빠뜨린 것이 있을 수도 있고, 유용하지 않아서 익히지 않은 것도 있고, 제가 잘 모르거나 잘 못써서 적지 않은 것이 있을 수도 있습니다. 

🧚 본인만의 유용한 단축키가 있으신가요?

---

다음 동일 단어 선택 : cmd + D
현재 파일 내 동일 단어 모두 선택 : cmd + shift + L
선택된 줄 삭제 : cmd + shift + K

현재 파일 내 검색 : cmd + F
전체 파일 검색 : cmd + shift + F
전체 파일 검색 및 바꾸기 : cmd + shift + H

최근 열린 파일 목록 열기 및 파일명 검색 : cmd + P
설정 검색 : cmd + shift + P

파일 탐색기 열기 : cmd + shift + E
소스 제어창 열기 : ctrl + shift + G

현재 탭 닫기 : cmd + W
마지막 닫은 탭 다시 열기 : cmd + shift + T
열린 탭 모두 닫기 : cmd + K, W
탭 이동 : cmd + opt + 좌/우 화살표

새 파일 생성 : cmd + N
새 윈도우 창 열기 : cmd + shift + N

현재 줄 위치 이동 : opt + 위/아래 화살표
현재 줄 복사 : opt + shift + 위/아래 화살표

포인터 맨 위/아래로 이동 : cmd + 위/아래 화살표
포인터 다중 열 선택 : cmd + opt + 위/아래 화살표
포인터 이전 위치로 이동 : ctrl + -
포인터 다음 위치로 이동 : ctrl + shift + -

코드 블럭 축소 : cmd + opt + [
코드 블럭 확대 : cmd + opt + ]

P.S.
요즘 작업은 Cursor IDE로 하고 있지만, VSCode를 fork했기 때문에 동일한 단축키를 사용하고 있습니다. AI 용 단축키는 아래와 같이 사용 중입니다.
Cursor Edit AI : cmd + opt + K
Cursor Chat AI : cmd + opt + L
