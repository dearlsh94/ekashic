# LinkedIn Writing Rules

## Format
- 500-800자 (한글 기준)
- 3줄 요약으로 시작 (Hook)
- 줄바꿈으로 가독성 확보

## Tone & Style
- 전문적이면서 친근함
- 결론부터 말하는 두괄식
- 개발자다운 비유와 예시 활용
- 짧은 문장, 명확한 전달
- 독자와의 공감대 형성

## Writing Principles
1. **Hook First**: 첫 문장으로 관심을 끌 것
2. **Concrete Examples**: 추상적인 이야기보다 구체적 사례
3. **Actionable Insight**: 읽고 바로 적용할 수 있는 내용
4. **Personal Voice**: 경험에서 우러나온 진정성

## Structure
```
[Hook - 1~2문장]
관심을 끄는 질문이나 도발적인 문장

[Context - 2~3문장]
왜 이 이야기를 하는지 배경

[Insight - 3~5문장]
핵심 메시지와 구체적 사례

```
