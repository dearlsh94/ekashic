# Blog Writing Rules

## Format
- 1500-3000자
- 마크다운 문법 활용
- 각 문단은 가능한 7 문장 이내

## Tone & Style
- 친근하 솔직하고 사려깊음.
- 문학적 비유 사용

## Writing Principles
1. **Hook First**: 첫 문장으로 관심을 끌 것
2. **Concrete Examples**: 추상적인 이야기보다 구체적 사례
3. **Actionable Insight**: 읽고 바로 적용할 수 있는 내용
4. **Personal Voice**: 경험에서 우러나온 진정성

## Structure
```
# 제목

## 서론
- 동기
- 이 글에서 다룰 내용 예고

## 본론

## 결론
- 핵심 요약
- 내용 정리
- 회고
```

## Essay Posts
- 개인 경험 기반
- 감정과 생각 솔직하게
